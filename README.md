## Installation
+ Clone Repo
+ Make sure `stow` is installed, and then run `./setup` (make sure u will update STOW_DIRECTORIES in ./setup and .zshrc)
+ Install neovim
+ Open it up and go to Command Mode and run `PlugInstall`
